# Pace — 制作プロジェクト管理

3〜4ヶ月スパンの制作プロジェクト（書籍・教材・コンテンツ）を毎日回すための管理ツール。

**👉 使う： https://shiraha2001.github.io/pace/**

## 特徴
- 単一HTML・インストール不要・無料
- データは各自のブラウザ内（localStorage）に保存 — サーバーには何も送信されません
- バーンダウンチャート／今日のノルマ自動計算／連続記録
- 工程テンプレート（書籍制作・標準フロー同梱）を新プロジェクトへワンタップ展開
- 「今日の報告」テキストを自動生成 → コピーしてLINE/Slackへ

## データについて
- ブラウザごとに独立保存（端末をまたぐ同期はありません）
- 右上「⤓」からJSON書き出し／読み込みでバックアップ・引っ越しができます
